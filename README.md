# 🎮 АДСКИЕ ШАХМАТЫ 

Шахматы, но с **пиздецом**. Ваши фигуры будут тупить, рандомить и оскорблять вас.

## 🚀 Особенности

- 🎲 **30% шанс** что фигура откажется ходить
- 💬 **Тролль-сообщения** после каждого хода
- 🎰 **Случайные события** (ферзь может уйти к другому игроку!)
- 🖼️ **Мемные ассеты** (замени картинки на что угодно)
- 🔥 **Режим боли** (включён по умолчанию)

## 📦 Установка

### Вариант 1: Запустить из исходников
```bash
# Установи Python 3.8+
pip install -r requirements.txt
python main.py
