# 📱 Настройка для Android

## 📋 Пошаговая инструкция

### 1️⃣ Установи v2rayNG
- Скачай из [Google Play](https://play.google.com/store/apps/details?id=com.v2ray.ang)
- Или [GitHub](https://github.com/2dust/v2rayNG/releases) (последняя версия)

### 2️⃣ Импортируй конфиг
1. Открой v2rayNG
2. Нажми **+** (плюс вверху справа)
3. Выбери **Import config from clipboard**
4. Вставь ссылку `vless://...`
5. Нажми **✓** (галочка)

### 3️⃣ Настрой DNS (если не работает)
1. Нажми на **три точки** рядом с конфигом
2. Выбери **Edit**
3. Прокрути вниз до **DNS**
4. Впиши: `https://8.8.8.8/dns-query`
5. Нажми **✓**

### 4️⃣ Включи Fragment (для T2/Билайн)
1. Edit профиль → прокрути до **Fragment**
2. Включи переключатель
3. Параметры:
   - Packets: `tlshello`
   - Length: `10-20`
   - Interval: `10-20`
4. Сохрани

### 5️⃣ Подключись
1. Вернись на главный экран
2. Нажми **галочку внизу** (круглая кнопка)
3. Подожди 5-10 секунд
4. Вверху появится ключик 🔑

## 🔧 Дополнительные настройки

### Разрешения
- v2rayNG запросит разрешение на VPN → **OK**
- Разрешения на уведомления → **Разрешить** (чтобы не отключалось)

### Автозапуск
- Настройки Android → Приложения → v2rayNG
- Включи: **Автозапуск**, **Работа в фоне**
