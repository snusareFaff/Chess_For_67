# 🖥️ Настройка для Windows

## 📋 Шаг за шагом

### 1️⃣ Скачай клиент
- Windows 10/11: **Hiddify Next** или **v2rayN**
- Windows 7: **v2rayN 3.28**

### 2️⃣ Импортируй конфиг
1. Открой программу
2. Нажми **Import** / **Добавить** / **Ctrl+V**
3. Вставь ссылку (начинается на `vless://`)
4. Нажми **OK**

### 3️⃣ Настрой DNS (если не подключается)
**Для Hiddify:**
- Открой профиль → Settings → DNS
- Впиши: `https://8.8.8.8/dns-query`
- Сохрани

**Для v2rayN:**
- Настройки → Параметры → DNS
- Выбери: `Google DNS` или впиши `8.8.8.8`

### 4️⃣ Включи Fragment (если блокирует провайдер)
**Важно для T2 (Tele2), Билайн, МТС!**

**Hiddify:**
- Edit профиль → Advanced
- Включи **Fragment**
- Параметры:
  - Packets: `tlshello`
  - Length: `10-20`
  - Interval: `10-20`

**v2rayN:**
- Правой кнопкой на профиль → Edit Server
- Вкладка **Stream Settings**
- Включи **Fragment**

### 5️⃣ Подключись
1. Выбери профиль
2. Нажми **Connect** / **Подключиться**
3. Подожди 5-10 секунд
4. Проверь IP: [ipinfo.io](https://ipinfo.io)

---

## ⌨️ Горячие клавиши (v2rayN)
- **F1** — Подключиться/отключиться
- **F2** — Копировать ссылку
- **F3** — Обновить подписку
