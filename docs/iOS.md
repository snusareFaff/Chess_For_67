# 🍎 Настройка для iOS (iPhone/iPad)

## 📋 Streisand (рекомендую)

### 1️⃣ Установи
- Открой **App Store**
- Найди **Streisand**
- Установи (бесплатно)

### 2️⃣ Импортируй конфиг
1. Открой Streisand
2. Нажми **+** (Add Server)
3. Выбери **Import from URL**
4. Вставь ссылку `vless://...`
5. Нажми **Import**

### 3️⃣ Включи Fragment (для T2/Билайн)
1. Нажми на свой профиль → **Edit**
2. Прокрути до **Advanced**
3. Включи **Fragment**
4. Параметры:
   - Packets: `tlshello`
   - Length: `10-20`
   - Interval: `10-20`
5. **Save**

### 4️⃣ Настрой DNS
1. Edit профиль → **DNS**
2. Впиши: `https://8.8.8.8/dns-query`
3. **Save**

### 5️⃣ Подключись
1. Вернись назад
2. Нажми **Connect**
3. iOS спросит разрешение на VPN → **Allow**
4. Подожди 5-10 секунд
5. Сверху появится значок VPN

---

## 🦊 FoXray (альтернатива)

1. Установи из App Store
2. Открой → Import Config
3. Вставь ссылку
4. В настройках включи **Fragment**
5. Подключись
